# VBA - Visual Basic for Applications

Visual Basic for Applications é a implementação da linguagem _Visual Basic_ (VB) incorporado em todas as aplicações do _Microsoft Office_.

## Por que aprender?
1. Otimização de tarefas: Através de um simples código é possível reduzir consideravelmente as tarefas manuais.
2. Maior manutenibilidade: Implementações em VBA possuem maior legibilidade do que implementações diretas com funções em células. Logo é mais fácil rastrear uma macro VBA do que funções inerentes de uma determinada planilha.
3. Integração com outras ferramentas: O VBA viabiliza integrar um software com outro com transcrição e leitura de dados automatizados
4. Fácil aprendizado: a linguagem é simples e intuitiva, com nível de abstração bastante elevado com funções nativas bastante robustas.

## Começando
VBA pode ser programado em qualquer aplicação do pacote office, entretanto este tutorial foi realizado utilizando o Excel, pois é onde a linguagem foi implementada com maior robustez.

### Configurando o ambiente

```sh
Guia de arquivo > Opções > Personalizar Faixa de Opções > Marcar CheckBox de Desenvolvedor
```
